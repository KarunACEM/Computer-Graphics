# Smart Traffic System

A Pygame-based traffic light simulation for a Computer Graphics project.

## Features
- Animated vehicles moving through a four-way intersection
- Traffic lights with NS/EW phases and timers
- Timers displayed for both NS and EW lights
- One vehicle per lane, stopping at red lights
- Vehicles colored by direction for clarity

## How to Run
1. Make sure you have **Python 3.x** installed
2. Install Pygame if not already installed:
   ```bash
   pip install pygame
